## Project Overview

<u>VISION</u> is an image recognition pipeline designed to accelerate taxonomy classification by combining classical statistical approaches with cutting-edge neural networks and LLM techniques. The system bridges traditional manual labeling methods with automated deep learning, enabling scalable, accurate, and rapid image classification for client deliverables.

I led the design, prototyping, and implementation of the pipeline, delivering production-ready models that now form a core part of large-scale workflows.

### Technologies and Approach

**Hybrid Modeling & Taxonomisation:**

- Combined traditional forecasting methods (ARIMA) with neural networks and LLM-assisted approaches to enable one-shot taxonomisation of image datasets.
- Optimised labeling and data selection to maximize model accuracy, ensuring neural networks could learn effectively from limited examples.

**Data & Model Engineering:**

- Partnered with traditional image labeling teams to transition manual expertise into custom PyTorch models, improving both speed and dataset targeting.
- Built a prototype image blurrer and classifier in Python using Ultralytics‚Äô YOLOv8, streamlining preprocessing and detection.
- Implemented feedback loops to continuously enhance model accuracy and reliability in production workflows.

**Productionisation & Workflow Integration:**

- Models are now embedded into client-facing deliverables, supporting large-scale automation of image classification tasks.
